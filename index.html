Timestamp Microservice

By Mankirk

User Stories:
  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: September 20,1992)
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example usage:

https://timestampmicroservice-mankirk.c9users.io/31236187461
https://timestampmicroservice-mankirk.c9users.io/september 1,2030

Example output:

{ "unix": "31236187461", "natural": "November 1, 2959" }
{ "unix": 1914451200, "natural": "september 1,2030" }
